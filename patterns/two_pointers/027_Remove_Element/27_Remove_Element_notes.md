## ✅ 你这段代码：逻辑完全正确

只需要一个**语法级的小修正**（`if` 换行）：

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        k = 0
        for i in range(len(nums)):
            if nums[i] != val:
                nums[k] = nums[i]
                k += 1
        return k
```

✔ 时间复杂度：`O(n)`
✔ 空间复杂度：`O(1)`
✔ 完全符合题意 & Judge
